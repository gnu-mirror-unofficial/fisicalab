Paquetes deb para Ubuntu y Debian

Estos paquetes son gracias a Philippe Roussel, autor de SimpleAgenda.

Dependiendo del sistema y la arquitectura primero se debe agregar la URL correspondiente a la lista de repositorios:

Debian Wheezy i386 http://coyote.octets.fr/pub/gnustep/packages/wheezy/i386/
Debian Wheezy amd64 http://coyote.octets.fr/pub/gnustep/packages/wheezy/amd64/
Ubuntu Precise i386 http://coyote.octets.fr/pub/gnustep/packages/precise/i386/
Ubuntu Precise amd64 http://coyote.octets.fr/pub/gnustep/packages/precise/amd64/
Ubuntu Quantal i386 http://coyote.octets.fr/pub/gnustep/packages/quantal/i386/
Ubuntu Quantal amd64 http://coyote.octets.fr/pub/gnustep/packages/quantal/amd64/
Ubuntu Raring amd64 http://coyote.octets.fr/pub/gnustep/packages/raring/amd64/

Por ejemplo para Debian Wheezy amd64 se puede agregar la linea:

deb http://coyote.octets.fr/pub/gnustep/packages/wheezy/amd64/ ./

Y despues de actualizar la lista de paquetes ya podra instalar el paquete fisicalab.app. Si no le gusta la apariencia por defecto, puede instalar el tema Silver para una apariencia más adaptada (solo descomprima el paquete y copie el contenido en la ruta ~/GNUstep/Library/Themes de su carpeta personal). Una vez instalado el tema ejecute lo siguiente en una terminal:

defaults write NSGlobalDomain GSTheme Silver

defaults write NSGlobalDomain GSAppOwnsMiniwindow NO

defaults write NSGlobalDomain GSSuppressAppIcon YES
