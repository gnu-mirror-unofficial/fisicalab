Guia de instalación en Windows

Para la instalación se deben descargar los siguientes paquetes:

  • GNUstep MSYS System
  • GNUstep Core
  • FísicaLab 0.3.2

Una vez descargados, instale los paquetes en el orden en el que estan listados, no necesita cambiar ninǵuna opción. Instalada la aplicación, se puede acceder en Inicio -> Aplicaciones -> FisicaLab -> FisicaLab.

Si usted esta actualizando desde la versión 0.3.0, desinstale primero dicha versión. También es reccomendable desactivar el backend Cairo. Ejecute el Shell en Inicio -> Aplicaciones -> GNUstep -> Shell y escriba:

defaults delete NSGlobalDomain GSBackend

y presione ENTER. Esto le impedira ver la documentación, pero puede descarga esta en formato PDF en la sección Manual.

NOTA IMPORTANTE
El carácter @, necesario para utilizar factores de conversión, no puede ingresarse utilizando la combinación Alt-64. Este debe ingresarse utilizando la tecla Alt Gr en combinación con alguna otra tecla alfanumérica (en teclados latinoamericanos la tecla alfanumérica para el 2).
