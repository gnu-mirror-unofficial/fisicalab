Guia de instalación en Windows (sólo 32 bits)

Para la instalación se deben descargar los siguientes paquetes:

  • GNUstep System.
  • GNUstep Core.
  • GSL.
  • FísicaLab-setup.

Una vez descargados, se instala primero GNUstep System, sin cambiar ninguna de las opciones. Seguidamente se instala GNUstep Core, tampoco se cambian las opciones. Estos intaladores crean una carpeta llamada GNUstep directamente en el disco duro. Es en esta carpeta donde debe instalarse GSL. Por último se instala FisicaLab-setup, sin cambiar las opciones. Ahora, en la ruta /GNUstep/Gnuwin32/bin se copian los archivos:

  • libgsl.dll
  • libgslcblas.dll

Y se pegan en la ruta /GNUstep/GNUstep/System/Tools. Finalmente, para utilizar la aplicación en español, se abre el Shell de GNUstep en Aplicaciones -> GNUstep -> Shell, y se escribe:

defaults write NSGlobalDomain NSLanguages "(Spanish)"

Se cierra el Shell, y ahora la aplicación se puede acceder en Aplicaciones -> GNUstep -> Apps -> FisicaLab.

Acerca del caracter @

En sistemas Windows el caracter @, necesario para utilizar factores de conversión, no puede ingresarse utilizando la combinación Alt-164. Este debe ingresarse utilizando la tecla Alt Gr o Alt en combinación con alguna otra tecla alfanúmerica.

Problemas conocidos en Windows

Actualmente GNUstep tiene aun algunos bugs en Windows, los cuales presentan algunos problemas. Sin embargo, ninguno de ellos es crítico. Estos se presentan a continuación:

  • Las etiquetas no funcionan (si se coloca el mouse sobre algun botón, se ve un recuadro pero no el texto de descripción).
  • El cursor no cambia (a la hora de agregar elementos a la pizarra, o mover elementos, el cursor no cambia como debería ser).
