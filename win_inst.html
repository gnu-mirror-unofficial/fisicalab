Guia de instalación en Windows

Si tiene instalada una version anterior de FísicaLab, asegurese de desinstalar esta junto con los paquetes de GNUstep.

Para la instalación se deben descargar los siguientes paquetes:

  • GNUstep MSYS System
  • GNUstep Core
  • GNUstep Cairo
  • FísicaLab 0.3.3

Una vez descargados, instale los paquetes en el orden en el que estan listados- Asegurese de activar el tema WinUXTheme al instalar el paquete Core, como se muestra en la siguiente imagen:

Las demás opciones no las modifique. Instalada la aplicación, se puede acceder en Inicio -> Aplicaciones -> FisicaLab -> FisicaLab.

Si olvido activar el tema, ejecute el Shell en Inicio -> Aplicaciones -> GNUstep -> Shell y escriba:

defaults delete NSGlobalDomain GSTheme WinUXTheme

y presione ENTER.

NOTA IMPORTANTE
El carácter @, necesario para utilizar factores de conversión, no puede ingresarse utilizando la combinación Alt-64. Este debe ingresarse utilizando la tecla Alt Gr en combinación con alguna otra tecla alfanumérica.

Problemas conocidos

  • El puntero del mouse no cambia como deberia hacerlo al agregar un elemento a la pizarra o al mover un elemento en esta.
