Guia de instalación en Windows (sólo 32 bits)

Para la instalación se deben descargar los siguientes paquetes:

  • GNUstep System
  • GNUstep Core
  • FísicaLab-0.2.1

Una vez descargados, se instala primero GNUstep System, sin cambiar ninguna de las opciones. Seguidamente se instala GNUstep Core, sin cambiar ninguna opción. Por último se instala FisicaLab-0.2.1-setup, sin cambiar las opciones. Instalada la aplicación, se puede acceder en Inicio -> Aplicaciones -> FisicaLab -> FisicaLab.

Notas importantes

  • El carácter @, necesario para utilizar factores de conversión, no puede ingresarse utilizando la combinación Alt-64. Este debe ingresarse utilizando la tecla Alt Gr en combinación con alguna otra tecla alfanumérica (en teclados latinoamericanos la tecla alfanumérica para el 2).
  • Debido a un bug de GNUstep en Windows el puntero del mouse no cambia como deberia hacerlo al agregar un elemento a la pizarra o al mover un elemento en esta.
