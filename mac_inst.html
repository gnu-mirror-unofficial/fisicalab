Guia de instalación en Mac OS

Usted necesita tener instalado Xcode para poder utilizar herramientas de compilación, así como SVN. Una vez instaladas estas herramientas, debe instalar los siguientes requisitos:

  • Las librerías GSL.
  • El paquete gnustep-make.
  • El framework Renaissance desde SVN:

    svn co http://svn.gna.org/svn/gnustep/libs/renaissance/trunk/ Renaissance

Consulte las intrucciones de instalación de cada uno de estos paquetes.

Ahora descarge y extraiga el paquete FísicaLa-Mac-0.2. Y en una terminal ubiquese en la carpeta extraida y corra primero el script de GNUstep (problamente en /Library/Makefiles/GNUstep.sh) con:

. /Library/Makefiles/GNUstep.sh

Luego compile la aplicación:

make

Y con privilegios de superusuario:

make install

Una vez instalado el programa puede ejecutarse con:

open FisicaLab
