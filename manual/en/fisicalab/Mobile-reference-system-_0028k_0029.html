2.2 Mobile reference system

A mobile reference system.

Equations:4
Data:
Name:Name of the mobile system.
Object:Name of the object inside the mobile system.
xsi:Initial X coordinate of the mobile system in reference to the stationary system.
ysi:Initial Y coordinate of the mobile system in reference to the stationary system.
vsx:Velocity in X of the mobile system.
vsy:Velocity in Y of the mobile system.
xof:Final X coordinate of the object in reference to the stationary system.
yof:Final Y coordinate of the object in reference to the stationary system.
vxof:Final velocity in X of the object in reference to the stationary system.
vyof:Final velocity in Y of the object in reference to the stationary system.
