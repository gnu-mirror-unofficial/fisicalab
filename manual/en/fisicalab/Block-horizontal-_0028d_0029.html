10.5 Block with horizontal movement

Block with horizontal movement. Allows only one horizontal friction force.

Equations:4
Data:
Name:Name of the block.
m:Mass of the block.
a:Acceleration of the block.
vi:Initial velocity.
vf:Final velocity.
d:Travelled distance.
Relative to:To default is sf meaning that the data is measured from a stationary reference system. But you can set a block, if the data is relative to that block.
