1.4 Messages

FisicaLab write a wide variety of messages in the text view when a problem is wrong. However, you always will see a message about the calculation’s status, as you can see in the following image:


The last line say "State = success", meaning that the calculation was successful. Any other status different as success, mean that or the set problem don’t have a solution, or an unexpected error occurred.
