Install on systems GNU/Linux, BSD, ...

FísicaLab 0.2 is available in the following distros:
  • Gentoo
  • OpenBSD

Donwload the package:

  • FisicaLab-0.2

FísicaLab need GNUstep (See Guide to install GNUstep) and GSL libraries (the dev headers too). With these requirements, on a terminal cd to the source directory and execute make and, with root permissions, make install. It is expected that the GSL headers are in the path /usr/include and libraries in /lib/gsl. Otherwise, modify the file GNUmakefile.preamble according to the proper paths. To run the application type openapp FisicaLab.

If you aren't using the WindowMaker desktop, we recommend install the IconManager tool for a better experience (see the INSTALL file for installation instructions). We also recommend install the theme Silver a more suited theme to these desktops. Just unzip the package and copy it in the ~/GNUstep/Library/Themes/ in your user folder. And set the theme on SystemPreferences.app, or in a terminal with:

. /usr/GNUstep/System/Library/Makefiles/GNUstep.sh

defaults write NSGlobalDomain GSTheme Silver

Or, if you wish that only FísicaLab use this theme:

. /usr/GNUstep/System/Library/Makefiles/GNUstep.sh

defaults write FisicaLab GSTheme Silver
