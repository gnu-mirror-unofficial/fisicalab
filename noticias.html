  • Marzo 12, 2013: FísicaLab 0.3.2 liberado. Esta versión corrige un problema con el menú contextual de los factores de conversión. Así como algunos otros cambios que se describen a continuación:
    • Liempieza de Warnings cuando se utiliza el compilador clang.
    • Añadido un script de configuración.
    • Renombrado de todas las clases al Inglés.

    Esta versión también esta disponible como binario para sistemas GNU/Linux de 32 bits. El paquete de código requiere las últimos paquetes estables de GNUstep. Los distintos paquetes pueden obtenerse en la página de Descargas.

  • Octubre 2, 2012: FísicaLab 0.3.0 liberado. Esta versión añade las siguientes características a la interfaz gráfica:
    • Los factores de conversión pueden agregarse a los datos desde un menú contextual.
    • Pueden realizarse operaciones matemáticas básicas en las entradas de la tabla, así como hacer uso de algunas funciones como sin(), cos(), hypot(), etc.
    • Un recuadro amarillo resalta el elemento de la pizarra que se esta editando.
    • Al dejar el puntero del ratón sobre algún elemento en la pizarra, la información de este se despliega en una etiqueta.
    • Se ha agregado un panel de preferencias que permite configurar el tamaño de la pizarra (en celdas), así como el tamaño de la fuente de las etiquetas.

    Para conocer más acerca de estas mejoras, consulte la documentación. En cuanto a los módulos se han hecho los siguientes cambios y añadidos:

    • Cinemática circular de partículas: Cambios menores en las entradas de algunos elementos. Por ejemplo el elemento Velocidad Angular que ahora tiene entradas tanto para la velocidad angular inicial como para la velocidad angular final.
    • Dinámica de partículas: Se ha eliminado el elemento Explosión. Ya que estos problemas pueden resolverse de forma más adecuada en el nuevo modulo añadido.
    • En los dos módulos de estática se ha agregado un elemento para relacionar ángulos.
    • Estática de cuerpos rígidos: Este módulo se ha extendido para poder resolver problemas de armaduras, también se han agregado elementos que representas vigas sometidas a dos fuerzas.
    • Nuevo módulo Dinámica circular de partículas.

    Para más detalles véase la documentación. Esta versión también esta disponible como binario para sistemas GNU/Linux de 32 bits y como instalador para Windows. Los distintos paquetes pueden obtenerse en la página de Descargas.

    Si desea compilar FísicaLab desde el código fuente, necesitara utilizar los paquetes en desarrollo de GNUstep.

  • Diciembre 31, 2010: Se descrube el primer bug en FísicaLab 0.2. Vea la página de bugs.
  • Noviembre 3, 2010: FísicaLab 0.2 esta disponible en los repositorios de las distribuciones Gentoo y OpenBSD.
  • Julio 6, 2010: Disponible FísicaLab 0.2. Esta nueva versión trae tres nuevos submódulos. Los cuales son:
    1. Cinemática del movimiento circular de partículas.
    2. Estática de cuerpos rígidos en 2D.
    3. Calor, calorimetría, gases ideales y dilatación.

    Además se han corregido algunos bugs y se han hecho varias mejoras, entre estas:

    1. Corregidos algunos factores de conversión incorrectos.
    2. Más mensajes para prevenir la combinación incorrecta de elementos.
    3. Mejoras para prevenir bloqueo del puntero del mouse a la hora de mover o agregar elementos a la pizarra.
    4. La versión para Windows tiene autodetección de lenguaje. Además de venir con las librerías de GSL incluidas. Y de utilizar decoración nativa para las ventanas y un tema especial para Windows.

    En la sección Descargas puede obtenerse esta nueva versión.
