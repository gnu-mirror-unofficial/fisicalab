GNUSTEP

Guia para instalar GNUstep en sistemas GNU/Linux

Los siguientes paquetes son requisitos para poder instalar GNUstep, deben instalarse las librerías y las cabeceras de desarrollo:

  • GCC
  • Compilador para Objective C
  • librería ffi
  • librería ssl
  • librería xml
  • librería xslt
  • librería jpeg
  • librería tiff4
  • librería png
  • librería ungif
  • librería cairo
  • librería gmp3
  • freetype
  • librería x11
  • librería xext
  • librería xft
  • librería xmu
  • librería cups
  • librería aspell y aspell-es

La forma más fácil de instalar GNUstep es con el paquete gnustep-startup. La última versión puede obtenerse aquí. Una vez que el archivo ha sido extraido, en una terminal vamos a la carpeta extraida y, con permisos de root:

./InstallGNUstep

Esto inicia la configuración de GNUstep. Una vez terminada la configuración, se presenta un resumen de la misma indicando algun error en caso de que falte alguna libraría. Si todo esta bien presionamos ENTER para continuar con la instalación. Terminada esta, solo falta asegurar el inicio de los servicios de GNUstep al arrancar el sistema. Para ello en el archivo rc.local, generalmente ubicado en la ruta /etc, agregamos el siguiente código antes de exit 0 (se necesitan permisos de root):

GNUSTEP_SYSTEM_ROOT=/usr/GNUstep/Local

if [ -f $GNUSTEP_SYSTEM_ROOT/Tools/gdomap ]; then

$GNUSTEP_SYSTEM_ROOT/Tools/gdomap

fi

Ahora, en el archivo .profile o .bash_profile de tu carpeta personal (observa que estos son archivos ocultos) necesitas agregar el siguiente código:

. /usr/GNUstep/System/Library/Makefiles/GNUstep.sh

gdnc

gpbs

make_services

Con esto realizado, al iniciar tu sesion los servicios gdomap, gdnc y gpbs deben estar corriendo. Tal vez desees instalar también la aplicación System Preferences para configurar GNUstep. Dicha aplicación se puede obtener aquí. Así como la aplicacion IconManager, muy útil en escritorios Gnome, KDE, Xfce, etc. Y que puede descargarse aquí.
