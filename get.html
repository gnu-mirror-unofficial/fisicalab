Downloading the Source Code

The source tarball of FisicaLab can be found at Savannah: http://download.savannah.gnu.org/releases/fisicalab/

Binaries

  • Universal binary for GNU/Linux distributions 32 bits
    • fisicalab-0.3.3-gnu-linux-32.tar.gz
  • Gentoo
    • Official Package
  • OpenBSD
    • OpenBSD Ports Package
  • FreeBSD
    • FreshPorts Package
  • Debian and Ubuntu
    • Add the corresponding Unofficial Repository (the available version of fisicalab isn't up to date)
  • MS Windows
    • FisicaLab Windows Installer. This package is not standalone, so you should first install the following prerequisites packages. Install these in the order that are listed.
      • GNUstep MSYS System
      • GNUstep Core
      • GNUstep Cairo
      Be sure to active the theme WinUXTheme when install the Core package. If you miss activate the theme at installation, run the Shell at Start -> Applications -> GNUstep -> Shell and write: defaults write NSGlobalDomain GSTheme WinUXTheme and hit RETURN. Keep in mind that in Windows systems you can't write the character @, needed to use conversion factors, with the combination Alt-64. You need write this character with the key Alt or Shift in combination with another alfanumeric key.
