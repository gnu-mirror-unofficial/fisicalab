Obtaining GNU FisicaLab

Downloading the Source Code

The source tarball of FisicaLab can be found on the main GNU ftp server: https://ftp.gnu.org/gnu/fisicalab/ (via HTTP) and ftp://ftp.gnu.org/gnu/fisicalab/ (via FTP). It can also be found on the GNU mirrors; please use a mirror if possible.

Source tarball for Mac OS X fisicalab-mac-0.3.5-1.tar.gz. See the file INSTALL for instructions.

Binaries

  • Universal binary for GNU/Linux i686 distributions
    • fisicalab-0.3.5-gnulinux-i686.tar.gz
  • Universal binary for GNU/Linux x86_64 distributions
    • fisicalab-0.3.5-gnulinux-x86_64.tar.gz
  • GNU/Linux and BSD Systems
    • Executable versions of FisicaLab for GNU/Linux and BSD systems are provided by the individual distributions. Distributions known to package FisicaLab include: Gentoo and OpenBSD. These packages are created by volunteers. The delay between a FisicaLab source release and the availability of a package for a particular GNU/Linux or BSD distribution varies. The FisicaLab project has no control over that process.
  • MS Windows
    • FisicaLab 0.3.5.1 Windows Installer. This package is self contained, so you don't need install any third party packages. If you have a previous version of FisicaLab, uninstall first all the related packages before install this new. Keep in mind that in Windows systems you can't write the character @, needed to use conversion factors, with the combination Alt-64. You should write this character with the key Alt or Shift in combination with another alfanumeric key.
