Obteniendo GNU FísicaLab

Descargar el Código Fuente

El código fuente de FísicaLab se puede encontrar en Savannah: http://download.savannah.gnu.org/releases/fisicalab/

Binarios

  • Binario universal para sistemas GNU/Linux de 32 bits
    • fisicalab-0.3.3-gnu-linux-32.tar.gz
  • Gentoo
    • Paquete Oficial
  • OpenBSD
    • Paquete OpenBSD Ports
  • FreeBSD
    • Paquete FreshPorts
  • Debian y Ubuntu
    • Agregue el correspondiente Repositorio No Oficial (los paquetes de fisicalab pueden no estar actualizados a la última versión)
  • MS Windows
    • Instalador de FísicaLab para Windows. Este paquete no es independiente, por lo que primero debe instalar los siguientes requisitos. Instale estos en el orden en que están listados.
      • GNUstep MSYS System
      • GNUstep Core
      • GNUstep Cairo
      Asegúrese de activar el tema WinUXTheme cuando instale el paquete Core. Si olvida hacerlo, arranque el Shell en Inicio -> Aplicaciones -> GNUstep -> Shell y escriba: defaults write NSGlobalDomain GSTheme WinUXTheme y presione RETURN. Tenga presente que en los sistemas Windows no puede escribir el carácter @, utilizado en las conversiones, mediante la combinación Alt-64. En su lugar, debe utilizar la tecla Alt o Shift en combinación con una tecla alfanumérica.
