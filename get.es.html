Obteniendo GNU FísicaLab

Descargar el Código Fuente

El código fuente de FísicaLab puede encontrarse en el servidor principal ftp de GNU: https://ftp.gnu.org/gnu/fisicalab/ (vía HTTP) y ftp://ftp.gnu.org/gnu/fisicalab/ (vía FTP). También puede encontrarse en los espejos GNU; por favor utilice un espejo de ser posible.

Código fuente para Mac OS X fisicalab-mac-0.3.5-1.tar.gz. Vea las instrucciones en el archivo INSTALACION.

Binarios

  • Binario universal para sistemas GNU/Linux i686
    • fisicalab-0.3.5-gnulinux-i686.tar.gz
  • Binario universal para sistemas GNU/Linux x86_64
    • fisicalab-0.3.5-gnulinux-x86_64.tar.gz
  • Paquetes binarios para sistemas GNU/Linux y BSD
    • Algunas distribuciones ofrecen paquetes binarios, sabemos que las distribuciones Gentoo y OpenBSD lo hacen. El proyecto FísicaLab no tiene control sobre estos paquetes, por lo que pueden no estar actualizados a la última versión disponible.
  • MS Windows
    • Instalador de FísicaLab 0.3.5.1 para Windows. Este paquete es independiente, por lo que no necesita instalar paquetes de terceros. Si tiene instalada una versión anterior de FisicaLab, asegurese the desinstalar todos los paquetes relacionados antes de instalar esta nueva versión. Tenga presente que en los sistemas Windows no puede escribir el carácter @, utilizado en las conversiones, mediante la combinación Alt-64. En su lugar, debe utilizar la tecla Alt o Shift en combinación con una tecla alfanumérica.
