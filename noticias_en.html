News

  • October 22, 2011: There is available a package of FisicaLab 0.2.1 for Mac OS. You can download it at Download page. Please report any problem with it at discussion list.
  • June 16, 2011: FísicaLab 0.2.1 released. This is a review version that brings improvements to work with the latest GNUstep packages and take advantage of new additions. Among these improvements are:
    • Fixed bugs.
    • Improved application performance.
    • Menu in window to GNU/Linux, *BSD, ...
    • Corrections to the menu in Windows 7.
    • Reimplementation of the help panel according to changes in GNUstep.

    You can get this new version on Downloads section.

  • December 31, 2010: Discovered the first bug on FísicaLab 0.2. See the bugs page.
  • November 3, 2010: FísicaLab 0.2 is available on the repositories of distros Gentoo and OpenBSD.
  • July 6, 2010: FísicaLab 0.2 released. This version comes with three new submodules:
    1. Kinematics of circular motion of particles.
    2. Static or rigid bodies in 2D.
    3. Heat, calorimetry, ideal gas and expansion.

    There are many improvements and fixed bugs:

    1. Fixed some incorrect conversion factors.
    2. More messages to prevent invalid combination of elements.
    3. Improvements to prevent blockage of the mouse pointer when moving or adding items to the board.
    4. Windows version comes with GSL libraries. And with an special theme to Windows.

    You can get this new version on Downloads section.

  • May 24, 2010: We discover a new bug in the stable release. See the section bugs.
  • January 12, 2010: Unfortunately FísicaLab 0.2 could not be released in late 2009. The design issues of the submodules have taken longer than expected. But I hope to release the next version during the first quarter of this year.
  • October 25, 2009: FísicaLab will be presented in CGSOL (Guatemalan Congress of Free Software) to be held from 26 to 28 November in Chimaltenango, Guatemala.
