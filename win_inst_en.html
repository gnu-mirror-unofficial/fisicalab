Guide to install FísicaLab on Windows (only 32 bits)

You need donwload the following packages:

  • GNUstep System.
  • GNUstep Core.
  • GSL.
  • FísicaLab-setup.

Now install first GNUstep System, without change the options. Second install GNUstep Core, without change the options too. This make a folder call GNUstep on the hard disck. Inside this folder, install GSL. Now install FisicaLab-setup, without change the options. Finally, in the path /GNUstep/Gnuwin32/bin copy the files:

  • libgsl.dll
  • libgslcblas.dll

And paste these on the path /GNUstep/GNUstep/System/Tools. Now, you can launch the app from Applications -> GNUstep -> Apps -> FisicaLab.

About the character @

On Windows systems you can't write the character @, needed to use conversion factors, with the combination Alt-164. You need write this character with the key Alt Gr or Alt in combination with another alfanumeric key.

Know problems on Windows

Currently GNUstep have some bugs on Windows, these cause some problems, none critical. Here there is a list of these problems:

  • Tooltips don't work.
  • The cursor don't change when you add or move elements on the chalkboard.
