Guide to install on Windows (only 32 bits)

You need donwload the following packages:

  • GNUstep System
  • GNUstep Core
  • FísicaLab-0.2.1

Now install first GNUstep System, without change the options. Second install GNUstep Core, without change the options too. Now install FisicaLab-0.2, without change the options. Then you can launch the app from Start -> Applications -> FisicaLab -> FisicaLab.

Notes

  • On Windows systems you can't write the character @, needed to use conversion factors, with the combination Alt-64. You need write this character with the key Alt Gr in combination with another alfanumeric key.
  • The mouse pointer don't change when you add or move elements on the chalkboard, as must be. This is a bug on GNUstep.
