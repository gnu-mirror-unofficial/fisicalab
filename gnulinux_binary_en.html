Install on systems GNU/Linux 32 bits

FísicaLab 0.2.1 is available in the following distros:
  • Gentoo

This is a binary package that has not been tested widely. But is the most easy way to get FisicaLab on GNU/Linux. Download the package:

  • GNU/Linux Binary

Extract the package and in a shell:

cd fisicalab-0.2.1-glinux32

./fisicalab

The first time you launch FisicaLab the cursor don't change as expected. This isn't a big problem, but is solved if you restart the app. More info at files INSTALL and README. Please report any problem at discussion mailing list here.
