Instalación en sistemas GNU/Linux, BSD, ...

Descarga el siguiente paquete:

  • FisicaLab-0.3.3

FísicaLab necesita de GNUstep y de las librerías GSL (incluidas las cabeceras de desarrollo). Cumplidos estos requisitos, en una terminal nos ubicamos en la carpeta del código y corremos, de ser necesario, el script de GNUstep:

. /usr/GNUstep/System/Library/Makefiles/GNUstep.sh

Luego ejecutamos:

./configure

make

Y con permisos de superusuario (puede ser necesario ejecutar nuevamene el script):

make install

La aplicación se corre con el comando openapp FisicaLab. Para poder utilizar FísicaLab en español, en una terminal ejecutamos:

. /usr/GNUstep/System/Library/Makefiles/GNUstep.sh

defaults write NSGlobalDomain NSLanguages "(Spanish)"

Si no se esta utilizando el escritorio WindowMaker, recomendamos el tema Silver para una apariencia más adaptada. Una vez instalado el tema lo puede establecer mediante la aplicación SystemPreferences.app, o mediante una terminal con:

. /usr/GNUstep/System/Library/Makefiles/GNUstep.sh

defaults write NSGlobalDomain GSTheme Silver

También se recomienda la siguiente configuración:

defaults write NSGlobalDomain GSAppOwnsMiniwindow NO

defaults write NSGlobalDomain GSSuppressAppIcon YES

defaults write NSGlobalDomain NSMenuInterfaceStyle NSWindows95InterfaceStyle
