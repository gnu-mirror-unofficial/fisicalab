Instalación en sistemas GNU/Linux, BSD, ...

FísicaLab 0.2 esta disponible en los repositorios oficiales de las siguientes distribuciones:
  • Gentoo
  • OpenBSD

Descarga el siguiente paquete:

  • FisicaLab-0.2.1

FísicaLab necesita de GNUstep (Ver la Guía de instalación de GNUstep) y de las librerías GSL (incluidas las cabeceras de desarrollo). Cumplidos estos requisitos, en una terminal nos ubicamos en la carpeta del código y corremos primero el script de GNUstep:

. /usr/GNUstep/System/Library/Makefiles/GNUstep.sh

(en algunos sistemas puede ser . /usr/share/GNUstep/System/Library/Makefiles/GNUstep.sh dependiendo de como se instalo GNUstep)

Luego ejecutamos make y, con permisos de superusuario:

. /usr/GNUstep/System/Library/Makefiles/GNUstep.sh

make install

Se espera que las cabeceras de GSL esten en la ruta /usr/include y las librerías en /lib/gsl. De no ser así, debe modificarse el archivo GNUmakefile.preamble acorde a las rutas apropiadas. La aplicación se corre con el comando openapp FisicaLab. Para poder utilizar FísicaLab en español, en una terminal ejecutamos:

. /usr/GNUstep/System/Library/Makefiles/GNUstep.sh

defaults write NSGlobalDomain NSLanguages "(Spanish)"

Si no se esta utilizando el escritorio WindowMaker (es decir GNOME, KDE, ...), recomendamos el tema Silver para una apariencia más adaptada a estos otros escritorios. Simplemente descomprima el paquete y copielo en el directorio ~/GNUstep/Library/Themes/ de su carpeta personal. Y establezca el tema mediante la aplicación SystemPreferences.app, o mediante una terminal con:

. /usr/GNUstep/System/Library/Makefiles/GNUstep.sh

defaults write NSGlobalDomain GSTheme Silver

O, si desea que únicamente FísicaLab utilice este tema:

. /usr/GNUstep/System/Library/Makefiles/GNUstep.sh

defaults write FisicaLab GSTheme Silver

También recomendamos establecer la siguiente configuración para una mejor experiencia en estos esritorios (GNOME, KDE, ...), desde una terminal:

. /usr/GNUstep/System/Library/Makefiles/GNUstep.sh

defaults write NSGlobalDomain GSAppOwnsMiniwindow NO

defaults write NSGlobalDomain GSSuppressAppIcon YES
