Helping to make known the project

There are several ways to help the project. Some of these are listed below:

  1. Using FísicaLab and comment what you think about.
  2. By introducing the application to teachers and professors of physics courses. As the students of such courses.
  3. If you work or study in an educational intitución that use FísicaLab, notify me so I can make a list of these institutions on the website.
  4. If you master some language which is not available on FísicaLab, and would like to help in the translation into that language, please contact me.
